# ファクトチェック

このコマンドは、プレゼンテーション内の事実関係を検証します。

## 実行内容

指定されたスライドファイルに対して、以下の項目をファクトチェックしてください：

### 検証対象

1. **数値・統計データ**
   - 出典が明記されているか
   - データの年度・調査対象が記載されているか
   - 数値が妥当か（桁違いなどの明らかな誤りがないか）

2. **引用・フレームワーク**
   - 原典からの引用か（孫引きではないか）
   - 引用が正確か
   - 著者名・書籍名が正しいか

3. **技術仕様**
   - 公式ドキュメントで確認できるか
   - バージョンが明記されているか
   - 正式名称を使用しているか

4. **人名・組織名**
   - 正式名称か
   - 肩書きが正しいか

5. **年号・日付**
   - 年号が正しいか
   - 「最近」「近年」などの曖昧表現がないか
   - 現在形・過去形の使い分けが正しいか

## 出力形式

```markdown
## ファクトチェック結果

### 信頼性スコア: X.X/5.0点

| 評価項目 | スコア | 問題数 |
|---------|--------|--------|
| 数値・統計 | X/5 | XX件 |
| 引用・出典 | X/5 | XX件 |
| 技術仕様 | X/5 | XX件 |
| 人名・組織名 | X/5 | XX件 |
| 年号・日付 | X/5 | XX件 |

### 要確認事項（重大）

1. **スライドXX**: 「[引用]」
   - 問題: [...]
   - 確認方法: [...]

### 要確認事項（軽微）

1. **スライドXX**: 「[引用]」
   - 問題: [...]

### 出典が不足している箇所

1. **スライドXX**: 「[主張内容]」
   - 推奨: [出典の追加方法]

### 出典が適切に記載されている箇所

1. **スライドXX**: [良い例]
```

## 出典の記載方法（参考）

### スライド内での表示

```markdown
<div style="position: absolute; bottom: 30px; right: 80px; font-size: 0.5em; color: #666;">
<a href="https://example.com/source">出典: 調査名（2025年）</a>
</div>
```

### 参考資料スライドでの表示

```markdown
## 参考資料

- [公式ドキュメント名](https://example.com/docs) - 2025年1月参照
- [調査レポート名](https://example.com/report) - 2024年版
```

## 使用方法

```
/fact-check [スライドファイルのパス]
```
